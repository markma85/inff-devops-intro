Server Pipeline

The Jenkins backend server pipeline is architected to utilize both Google Cloud Platform (GCP) and Amazon Web Services (AWS) for an efficient, hybrid-cloud CI/CD process. This architecture leverages the strengths of each platform, optimizing the build and deployment workflow for a .NET application with Dockerized components.
